AntrianApp - E-Antrian

Aplikasi antrian untuk pasien, dibangun dengan Kotlin + Firebase. 
Aplikasi ini dibuat khusus untuk pasien agar saat datang ke rumah sakit pasien tidak perlu lagi melakukan pendaftaran anrian, 
pasien langsung menunggu untuk pemeriksaan
